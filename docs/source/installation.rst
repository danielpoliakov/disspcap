=================
Build and install
=================

C++ shared library
******************

.. code:: bash

    $ git clone https://github.com/danieluhricek/disspcap
    $ cd disspcap
    $ make 


Python package
**************

.. code:: bash

    $ pip install disspcap

or 

.. code:: bash

    $ git clone https://github.com/danieluhricek/disspcap
    $ cd disspcap
    $ python setup.py install
